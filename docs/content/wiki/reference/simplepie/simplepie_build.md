## Description {#description}

```text
YYYYMMDDHHMMSS
```

Defines SimplePie's build ID.

## Availability {#availability}

Available from SimplePie 0.92-0.96; Since SimplePie 1.0

## Examples {#examples}

```php
echo 'This is ' . SIMPLEPIE_NAME . ', build ' . SIMPLEPIE_BUILD;
```
