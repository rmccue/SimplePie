SimplePie_Enclosure

SimplePie_Enclosure is used for handling all things related to enclosures and such (including Media RSS and iTunes RSS). This object is not typically called directly, get_enclosure() and get_enclosures() return SimplePie_Enclosure instances which allow us to access enclosure-related data. This class can be overloaded with set_enclosure_class().
