get_starttime()

Description

class SimplePie_Caption {
	get_starttime()
}

Returns the start time for the given caption. This mostly has relevance to Media RSS content.

Availability

  • Available since SimplePie 1.0.

Examples

$feed = new SimplePie();
$feed->set_feed_url('http://simplepie.org/blog/feed/');
$feed->init();
$feed->handle_content_type();
 
foreach ($feed->get_items() as $item)
{
	if ($enclosure = $item->get_enclosure())
	{
		if ($caption = $enclosure->get_caption())
		{
			echo '<p>' . $caption->get_text() . ' (' . $caption->get_language() . ')<br />';
			echo '<span class="footnote">' . $caption->starttime() . '&ndash;' . $caption->endtime() . '</span></p>';
		}
	}
}
